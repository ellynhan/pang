# PRD: 팡(PANG) 게임 — Mission 1

## 게임 개요

| 항목 | 내용 |
|------|------|
| 원작 | PANG (Mitchell Corporation, 1989) |
| 북미명 | Buster Bros. |
| 장르 | 아케이드 액션 |
| 플레이어 수 | 1~2인 (협력 플레이) |

팡은 전 세계 명소를 배경으로 하늘에서 튀어 다니는 풍선(버블)을 모두 제거하는 아케이드 게임이다.  
Mission 1의 배경은 **일본 후지산(Mount Fuji)**으로, 총 3개의 스테이지로 구성된다.

---

## 핵심 게임플레이

### 목표
스테이지에 존재하는 **모든 버블을 제거**하면 클리어. 버블에 닿으면 즉시 1미스.

### 플레이어 조작
- **좌우 이동**: 화면 바닥을 좌우로 이동
- **작살 발사**: 버튼을 누르면 위쪽으로 작살(와이어)을 쏘아 천장에 걸어둠
- 동시에 작살은 하나만 유지 (기본 무기 기준)

### 버블 분열 규칙
버블은 크기에 따라 4단계가 존재하며, 작살에 맞으면 **2개의 한 단계 작은 버블로 분열**된다.

```
대(Large) → 중(Medium) → 소(Small) → 극소(Tiny) → 소멸
```

- 버블이 클수록 튀는 높이가 높고 이동 속도가 느림
- 버블이 작을수록 낮게 튀고 이동 속도가 빠름
- 극소 버블을 맞추면 완전히 소멸

### 승패 조건
| 조건 | 결과 |
|------|------|
| 스테이지의 버블 전체 소멸 | 스테이지 클리어 |
| 버블에 접촉 | 1미스 (라이프 감소) |
| 라이프 소진 | 게임 오버 |

---

## Mission 1 구성 (후지산)

| 스테이지 | 특징 |
|----------|------|
| Stage 1 | 대형 버블 2개로 시작. 분열 패턴에 익숙해지는 튜토리얼 성격 |
| Stage 2 | 버블 수 증가, 지형(발판) 추가로 버블 경로가 복잡해짐 |
| Stage 3 | 다양한 크기의 버블이 혼재, 빠른 판단과 이동 요구 |

3스테이지를 모두 클리어하면 다음 Mission(스페인 등)으로 진행.

---

## 아이템 / 파워업

스테이지 중 버블을 제거하면 아이템이 드롭되며, 시간 안에 먹으면 획득한다.

| 아이템 | 효과 |
|--------|------|
| 더블 와이어 | 작살 2개 동시 발사 |
| 파워 와이어 | 더 빠르고 넓은 작살 |
| 벌칸 미사일 | 연속 총알 발사 (와이어 대체) |
| 방어막 | 버블 접촉 1회 무적 |
| 시계 | 화면의 버블 일시 정지 |
| 다이너마이트 | 화면 전체 버블 소멸 |
| 과일 / 별 | 점수 보너스 |
| 1UP | 라이프 추가 |

---

## 핵심 경험 목표

1. **직관적 조작** — 단순한 이동 + 발사 2버튼으로 즉시 이해 가능
2. **긴장감 있는 위기 관리** — 분열로 늘어나는 버블 수를 효율적으로 처리
3. **협력의 재미** — 2인 플레이 시 역할 분담과 호흡이 전략 요소가 됨
4. **점진적 난이도** — Mission 1은 기본 패턴 학습에 집중, 후반 Mission으로 갈수록 복잡도 증가
