# Mission 1 — 후지산 (Mount Fuji, Japan)

## 개요

| 항목 | 내용 |
|------|------|
| 배경 | 일본 후지산 |
| 스테이지 수 | 3 |
| 난이도 | 입문 (튜토리얼 성격) |
| 목적 | 기본 조작 및 버블 분열 패턴 학습 |

Mission 1은 게임 전체에서 가장 낮은 난이도를 제공하며, 플레이어가 버블 분열 메커니즘과 작살 조작에 익숙해질 수 있도록 설계된다.

---

## 스테이지별 구성

### Stage 1

| 항목 | 내용 |
|------|------|
| 초기 버블 | 대형(Large) 2개 |
| 지형 | 평지 (장애물 없음) |
| 난이도 | 매우 쉬움 |

- 대형 버블 2개가 화면 좌우에서 천천히 이동.
- 지형 장애물이 없어 버블 경로가 단순하고 예측하기 쉬움.
- **목표**: 분열 → 소형 → 극소 순으로 처리하는 기본 흐름 체험.

**버블 분열 전개 예시**
```
대 x2
 → 중 x4
   → 소 x8
     → 극소 x16 → 전부 소멸
```

---

### Stage 2

| 항목 | 내용 |
|------|------|
| 초기 버블 | 대형 1개 + 중형 2개 |
| 지형 | 중간 발판 1~2개 추가 |
| 난이도 | 쉬움 |

- 발판 위아래로 버블이 통과하며 예측이 조금 어려워짐.
- 중형 버블이 초반부터 등장하여 분열 시 소형 버블의 빠른 이동 속도에 대응해야 함.
- **목표**: 발판을 활용한 위치 선정, 소형 버블의 빠른 이동 대응 연습.

---

### Stage 3

| 항목 | 내용 |
|------|------|
| 초기 버블 | 중형 2개 + 소형 2개 |
| 지형 | 발판 2~3개, 높낮이 차이 있음 |
| 난이도 | 보통 |

- 처음부터 소형 버블이 포함되어 빠른 반응이 요구됨.
- 버블 총 개수가 분열 후 최대 12개 이상으로 늘어날 수 있어 화면 장악이 중요.
- 아이템 드롭 확률이 Stage 1~2보다 높아, 더블 와이어 등 무기 아이템 활용 기회가 생김.
- **목표**: 다수의 버블 동시 관리, 우선 처리 순서 판단.

---

## 난이도 설계 원칙

| 요소 | Mission 1 수준 |
|------|----------------|
| 버블 수 | 적음 (최대 분열 후 ~16개) |
| 버블 속도 | 느림~보통 |
| 지형 복잡도 | 단순 |
| 제한 시간 | 여유 있음 |
| 아이템 출현 빈도 | 보통 |

---

## 클리어 조건 및 실패 패턴

**클리어**: Stage 1 → Stage 2 → Stage 3 순으로 모든 버블 제거 시 Mission 2 진행.

**주요 실패 패턴**
- 대형 버블을 맞힌 후 분열된 중형 버블의 이동 방향을 예측하지 못해 접촉
- 소형·극소 버블이 몰릴 때 도망갈 공간 부족으로 접촉
- 작살 발사 타이밍을 놓쳐 버블이 플레이어 쪽으로 낙하
