# 게임 룰 상세

## 기본 목표

스테이지에 등장하는 **모든 버블을 작살로 제거**하면 클리어.  
버블 또는 적 오브젝트에 접촉하면 즉시 1미스 처리된다.

---

## 플레이어 조작

| 입력 | 동작 |
|------|------|
| 좌/우 방향키 | 플레이어 좌우 이동 |
| 공격 버튼 | 위 방향으로 작살(와이어) 발사 |

- 작살은 천장에 닿을 때까지 유지되며, 그 자리에 고정된 상태로 버블을 차단한다.
- 기본 무기는 동시에 1개의 작살만 유지 가능.
- 작살이 천장에 닿거나 버블에 맞으면 사라지고 다시 발사할 수 있다.

---

## 버블 시스템

### 크기 단계

버블은 총 4단계 크기로 구분되며, 작살에 맞을 때마다 2개의 한 단계 작은 버블로 분열한다.

```
대(Large) → 중(Medium) → 소(Small) → 극소(Tiny) → 소멸
```

### 버블 물리 특성

| 크기 | 튀는 높이 | 이동 속도 | 위험도 |
|------|-----------|-----------|--------|
| 대(Large) | 높음 | 느림 | 낮음 |
| 중(Medium) | 중간 | 보통 | 보통 |
| 소(Small) | 낮음 | 빠름 | 높음 |
| 극소(Tiny) | 매우 낮음 | 매우 빠름 | 매우 높음 |

- 버블은 좌우로 이동하며 바닥과 벽에 부딪히면 방향이 바뀐다.
- 분열 시 두 버블은 각각 좌·우 방향으로 튀어 나간다.

---

## 아이템 시스템

버블을 제거하면 아이템이 드롭되고, 일정 시간 안에 플레이어가 접촉하면 획득한다.

### 무기 아이템

| 아이템 | 효과 | 지속 |
|--------|------|------|
| 더블 와이어 | 작살 2개 동시 발사 | 스테이지 종료까지 |
| 파워 와이어 | 더 빠르고 폭이 넓은 작살 | 스테이지 종료까지 |
| 벌칸 미사일 | 연속 총알 발사 (와이어 대체) | 스테이지 종료까지 |

### 보조 아이템

| 아이템 | 효과 |
|--------|------|
| 방어막 | 버블 접촉 1회 무적 처리 |
| 시계 | 화면 내 모든 버블 일시 정지 |
| 다이너마이트 | 화면 내 모든 버블 즉시 소멸 |
| 과일 / 별 | 점수 보너스 |
| 1UP | 라이프 1 추가 |

---

## 라이프 & 게임 오버

- 초기 라이프: 3개
- 버블 접촉 시 1라이프 감소, 해당 스테이지 처음부터 재시작
- 라이프 0 → 게임 오버

---

## 점수 체계

| 행동 | 점수 |
|------|------|
| 버블 소멸 (크기별 상이) | 크기가 클수록 고점수 |
| 아이템 획득 | 종류별 보너스 점수 |
| 스테이지 클리어 | 클리어 보너스 |

---

## 스테이지 진행 흐름

```
Stage 시작
   ↓
버블 전체 제거
   ↓
스테이지 클리어 → 다음 Stage
   ↓ (미션 내 마지막 Stage 클리어)
다음 Mission 이동
```

버블에 접촉하면 현재 Stage 처음으로 되돌아가며 라이프가 1 감소한다.
