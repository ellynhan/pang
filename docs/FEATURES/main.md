# 메인 화면 구성

## 화면 레이아웃

게임 시작 시 표시되는 첫 진입 화면으로, 플레이어가 게임을 시작하거나 종료할 수 있는 허브 역할을 한다.

```
┌─────────────────────────────────┐
│                                 │
│            PANG                 │  ← 타이틀 로고
│                                 │
│                                 │
│      ▶ [ GAME  START ]          │  ← 선택 중인 항목 (커서 표시)
│        [   종   료   ]          │
│                                 │
│           HIGH SCORE            │
│             00000               │
│                                 │
└─────────────────────────────────┘
```

## 구성 요소

| 요소 | 설명 |
|------|------|
| 타이틀 로고 | "PANG" 로고 표시, 화면 상단 중앙 배치 |
| GAME START | 게임 시작 버튼 |
| 종료 | 게임 종료 버튼 |
| HIGH SCORE | 세션 내 최고 점수 표시 |

## 메뉴 조작

| 입력 | 동작 |
|------|------|
| ↑ / ↓ 방향키 | 메뉴 항목 간 이동 (위아래 순환) |
| Enter | 현재 선택된 항목 실행 |
| 마우스 클릭 | 해당 항목 바로 실행 |

- 화면 진입 시 첫 번째 항목(GAME START)이 기본 선택 상태
- 선택된 항목은 커서(▶) 및 강조 스타일로 시각적으로 구분
- 방향키는 목록 끝에서 반대쪽 끝으로 순환

## 동작 흐름

1. 게임 진입 시 타이틀 화면 표시, GAME START 항목 기본 선택
2. ↑/↓ 또는 마우스로 항목 선택
3. Enter 또는 클릭으로 실행

## 화면 전환

- **GAME START 선택** → Mission 1 Stage 1 로드
- **종료 선택** → 종료 확인 후 앱 종료
